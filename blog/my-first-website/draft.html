My First Website!



08/25/2018

Hello World! This is my very first website! I coded it by hand using HTML, CSS, and JavaScript. While I do have some experience using jQuery and Bootstrap, I wanted to start out with a site that does not use any libraries, frameworks, or preprocessors. I built this site both to practice the web development skills that I have been learning recently, to document my learn-to-code journey through this blog, and to serve as a portfolio of the coding projects that I build.

My first experience using code was a few years ago, while I was in college, where I used some R and SQL while working for an institute that did research into International Relations scholarship. While I haven't continued to learn data science, I was surprised by how much I enjoyed using code. The ability to be able to manually tell a computer what to do and to have your code be read, understood, and to see the result just feels so powerful. I then took Khan Academy's intro to HTML and CSS course to learn the basics, just for fun!

About a year ago I started to realize that software development could be an actual career for me, even without a computer science degree, and started following freeCodeCamp's online curriculum. Unfortunately, I have had a little trouble making steady progress since then. I have recently been looking into more structured (and less free) bootcamps that would help prevent me from getting stuck in my progress.

A week ago I finished Lambda School's free web development mini bootcamp, which I highly recommend. The instructor did an excellent job explaining a lot of JavaScript fundamentals, which helped me to really understand a few concepts that I had seen before but not quite internalized. I also watched and followed along with Gordon Zhu's Watch and Code intro curriculum, which I also recommend for many of the same reasons. His examples provided the basis of the 'To-Do List' project in my portfolio.

Going forward, I am going to continue to look into coding bootcamps. However, while I do that I will use this site to hold myself accountable to continue to learn! Every weekend I will post a blog post detailing what I learned or did that week! I do plan on continuing to use freeCodeCamp's curriculum for the time being. Well, thank you to anyone who bothered to read my rambling post, and check back next week!
